# fag-zoo
Aplicação web para gestão de animais em zoológicos desenvolvida com ReactJS, NodeJS e banco de dados MySQL. Foram utilizadas nesta aplicação tecnologias como express para desenvolvimento de apis rest, autenticação por JWT, multer para upload de arquivos, material-ui para interface do front-end, componente ag-grid para renderização de tabelas e entre outras tecnologias.
